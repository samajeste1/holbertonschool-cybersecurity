# Identity Management

This module covers Linux identity and access management including user auditing, SSH hardening, password policies, and privilege management.

## Tasks

- `0-audit_uid.sh` - Detect unauthorized UID 0 accounts
- `1-audit_shells.sh` - Identify service accounts with interactive shells
- `2-audit_groups.sh` - Detect users in dangerous groups
- `3-harden_ssh.sh` - Harden SSH configuration
- `4-pw_policy.sh` - Enforce PAM password policy
- `5-audit_crypto.sh` - Detect weak MD5 password hashes
- `6-onboard.sh` - Secure user onboarding with SSH key
- `7-sudo_config.sh` - Configure least privilege sudo
